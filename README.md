# Booking Data Analysis Dashboard

This project is an interactive data analytics dashboard built using Streamlit to analyze booking data for multiple services.

## Features
- Interactive filters (Date, Service Type, Price)
- Booking trends visualization
- Monthly revenue analysis
- Peak booking hours analysis
- Top customer spending insights

## Technologies Used
Python
Pandas
Streamlit
Plotly

## Run the Project

pip install -r requirements.txt
streamlit run app.py
